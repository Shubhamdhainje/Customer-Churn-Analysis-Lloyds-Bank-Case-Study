## Customer Churn Analysis – Lloyds Bank Case Study
________________________________________________________________________________________________________
## Project Overview
This project focuses on analyzing customer behavior to identify patterns related to customer churn 
in a banking dataset inspired by a Lloyds Bank case study. The goal was to clean, integrate, and 
prepare the data for future machine learning models that can predict customer churn.
________________________________________________________________________________________________________
## Objectives
- Understand customer behavior and churn patterns
- Integrate multiple datasets using a common CustomerID
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Prepare a structured dataset for machine learning models
________________________________________________________________________________________________________
## Dataset Information
The dataset includes:
- Customer demographics
- Transaction history
- Online activity
- Customer service interactions
- Churn status
________________________________________________________________________________________________________
## Project Workflow

### 1. Data Integration
Merged multiple datasets using CustomerID.
________________________________________________________________________________________________________
### 2. Data Cleaning
Handled missing values and ensured data consistency.
________________________________________________________________________________________________________
### 3. Feature Engineering
Created new features such as:
- **TotalSpent**
- **ServiceInteractions**
________________________________________________________________________________________________________
### 4. Exploratory Data Analysis (EDA)
Used visualizations and statistical analysis to understand customer behavior patterns.
________________________________________________________________________________________________________
### 5. Data Preparation
Prepared the final dataset suitable for machine learning models to predict customer churn.
